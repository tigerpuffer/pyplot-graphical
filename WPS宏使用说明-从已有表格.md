# WPS表格宏 - 从已有表格导出JSON配置

## 功能简介

这个WPS.js宏可以从已有的WPS表格中读取数据,并通过交互式对话框配置各种选项,最后导出为JSON配置文件,供Python绘图工具使用。

## 主要特性

✅ **交互式配置向导** - 通过对话框逐步配置所有选项
✅ **图表类型选择** - 支持7种图表类型
✅ **X轴可选启用** - 可选择是否使用X轴数据
✅ **选项框配置** - 网格、图例、数值显示均用选项框配置
✅ **单元格选择** - 所有标签和数据通过鼠标选择单元格
✅ **多数据集支持** - 可以添加多个数据集
✅ **智能数据读取** - 自动读取单元格和区域的数据

## 安装步骤

### 方法一: 直接使用宏代码

1. 打开WPS表格
2. 按 `Alt + F11` 打开宏编辑器
3. 点击 `插入` -> `模块`
4. 将 `wps_read_table.js` 文件的内容复制粘贴到模块中
5. 保存并关闭宏编辑器

### 方法二: 加载宏文件(推荐)

1. 打开WPS表格
2. 点击 `开发工具` -> `宏` -> `浏览`
3. 选择 `wps_read_table.js` 文件
4. 点击 `打开` 加载宏

## 使用方法

### 快速开始(5步配置)

1. **打开WPS表格** - 确保你的数据表格已打开
2. **运行宏** - 按 `Alt + F8`,选择 `StartExportWizard`,点击运行
3. **跟随向导** - 按照对话框提示逐步配置
4. **选择单元格** - 用鼠标点击选择相应的单元格
5. **导出JSON** - 最后选择保存位置导出配置文件

### 详细配置步骤

#### 步骤 1/5: 选择图表类型

对话框会显示7种图表类型,请输入对应的编号:
```
1. 折线图
2. 柱状图
3. 散点图
4. 误差棒图
5. 直方图
6. 饼图
7. 雷达图
```

**支持的图表类型:**
- 折线图: 适合显示趋势和变化
- 柱状图: 适合比较不同类别的数据
- 散点图: 适合显示数据分布
- 误差棒图: 适合显示数据的不确定性
- 直方图: 适合显示数据分布频率
- 饼图: 适合显示比例关系
- 雷达图: 适合多维度数据比较

#### 步骤 2/5: 配置基本选项

通过选项框配置以下选项(输入1=是, 0=否):

- **启用X轴数据**: 是否使用X轴数据
  - 某些图表(如饼图、直方图)可能不需要X轴数据
  - 选择"否"时,将跳过X轴数据选择步骤

- **显示网格**: 是否在图表中显示网格线
  - 建议开启,便于读取数据

- **显示图例**: 是否显示图例
  - 多数据集时建议开启

- **显示数值**: 是否在图表上显示具体数值
  - 根据需要选择

#### 步骤 3/5: 选择标签单元格

系统会依次提示选择以下单元格:

1. **图表标题单元格** - 选择包含图表标题的单元格
   - 例如: 选择 A1 单元格,内容为"销售数据趋势"

2. **X轴标签单元格** - 选择包含X轴名称的单元格
   - 例如: 选择 A2 单元格,内容为"月份"

3. **Y轴标签单元格** - 选择包含Y轴名称的单元格
   - 例如: 选择 B1 单元格,内容为"销售额(万元)"

**操作方法:**
- 点击"确定"后,对话框会暂时隐藏
- 用鼠标点击选择相应的单元格
- 选择完成后,按回车或点击对话框的"确定"按钮

#### 步骤 4/5: 选择X轴数据(如果启用)

如果在步骤2中选择了"启用X轴数据",则需要选择X轴数据所在的单元格区域。

**选择方式:**
- 可以选择单个单元格(如果X轴数据在一个单元格中,用逗号分隔)
- 也可以选择多个单元格(每个单元格一个X轴值)

**示例:**
- 方式1: 选择 A3 单元格,内容为"1月,2月,3月,4月"
- 方式2: 选择 A3:A6 区域,每个单元格分别包含"1月","2月","3月","4月"

#### 步骤 5/5: 配置数据集

可以添加一个或多个数据集。每次添加数据集时,需要选择:

1. **数据集标签单元格** - 数据集的名称
   - 例如: 选择 B2 单元格,内容为"2023年"

2. **Y轴数据单元格区域** - Y轴的数值数据
   - 可以选择单个单元格(逗号分隔)或多个单元格
   - 例如: 选择 B3:B6 区域,包含数值数据

3. **误差数据单元格区域(可选)**
   - 系统会询问是否包含误差数据
   - 点击"确定"选择误差数据区域
   - 点击"取消"跳过(某些图表类型不需要误差数据)

4. **选择颜色(可选)**
   - 从对话框中选择数据集的颜色
   - 如果不选择,系统会自动分配颜色

**添加多个数据集:**
- 每添加完一个数据集,系统会询问是否继续添加
- 可以添加最多7个数据集
- 每个数据集可以选择不同的颜色

## 数据格式要求

### 单元格数据格式

**数值数据:**
- 如果数据在一个单元格中,必须用**逗号**分隔
- 例如: `10,20,30,40,50`
- 不要使用空格或其他符号作为分隔符

**多单元格数据:**
- 可以选择多个连续的单元格
- 每个单元格一个数据值
- 系统会自动读取并合并

**数据类型:**
- 所有数据应为数值类型
- X轴可以是字符串(类别)或数值
- Y轴必须是数值

### 数据长度要求

- **Y轴数据长度**必须与**X轴数据长度**一致
- 如果使用多个单元格,X轴和Y轴的单元格数量应该相同
- 饼图和直方图对X轴数据长度没有严格要求

## 导入到Python绘图工具

1. 打开Python绘图工具
2. 点击 `📥 导入配置` 按钮
3. 选择刚才导出的JSON文件

5. 点击 `🚀 一键绘制图表` 即可生成图表

## 高级技巧

### 批量处理

- 如果有多个类似格式的表格,可以重复使用相同的配置流程
- 建议将同类数据放在相似的位置,便于快速选择

### 数据准备建议

- 将标题放在表格的左上角
- 将标签放在数据区域的上方或左侧
- 保持数据的连续性,避免空行和空列
- 使用清晰的命名,方便识别

### 颜色选择

建议的颜色顺序:
1. 蓝色 (blue) - 主数据
2. 橙色 (orange) - 对比数据
3. 绿色 (green) - 正面数据
4. 红色 (red) - 负面数据或警告
5. 紫色 (purple) - 特殊数据

## 常见问题

### Q: 运行宏时提示"请先打开一个WPS表格文件"?
A: 确保已经打开包含数据的WPS表格文件。

### Q: 选择单元格时看不到对话框?
A: 对话框在等待你选择单元格,请用鼠标点击表格中的相应单元格,然后按回车或点击对话框的"确定"按钮。

### Q: 数据导入后图表显示不正确?
A: 检查以下几点:
- 数据格式是否正确(数值用逗号分隔)
- Y轴数据长度是否与X轴数据长度一致
- 是否选择了正确的单元格区域

### Q: 可以跳过某些步骤吗?
A:
- 步骤1-3是必需的
- 如果在步骤2选择"不启用X轴",步骤4会自动跳过
- 步骤5中可以只添加一个数据集

### Q: 如何修改已添加的数据集?
A: 目前不支持修改已添加的数据集,如果配置错误,请重新运行宏。

### Q: 支持哪些颜色的选择?
A: 支持10种颜色: blue, orange, green, red, purple, brown, pink, cyan, yellow, black

### Q: 误差数据是必需的吗?
A: 不是必需的。只有某些图表类型(如误差棒图)需要误差数据。系统会询问是否包含误差数据。

## 函数列表

- `ShowHelp()` - 显示帮助信息

## 注意事项

1. **表格格式**: 确保表格数据格式规范,避免合并单元格影响数据读取
2. **数据类型**: 数值数据不要包含文本格式
3. **分隔符**: 数值数据必须用逗号分隔,不要使用空格
4. **数据长度**: 注意X轴和Y轴数据长度的一致性
5. **单元格选择**: 选择单元格时要准确,避免选择错误
