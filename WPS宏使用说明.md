# WPS表格宏 - 导出JSON配置文件使用说明

## 功能简介

这个WPS.js宏可以将WPS表格中的数据导出为JSON配置文件,方便导入到Python绘图工具中进行可视化绘图。

## 安装步骤

### 方法一: 直接使用宏代码

1. 打开WPS表格
2. 按 `Alt + F11` 打开宏编辑器
3. 点击 `插入` -> `模块`
4. 将 `wps_export_config.js` 文件的内容复制粘贴到模块中
5. 保存并关闭宏编辑器

### 方法二: 加载宏文件(推荐)

1. 打开WPS表格
2. 点击 `开发工具` -> `宏` -> `浏览`
3. 选择 `wps_export_config.js` 文件
4. 点击 `打开` 加载宏

## 使用方法

### 步骤1: 创建数据模板

1. 在WPS表格中按 `Alt + F8` 打开宏对话框
2. 选择 `CreateTemplate` 函数
3. 点击 `运行`

这将自动创建两个工作表:
- **基本设置**: 包含图表的基本配置信息
- **数据集**: 包含各个数据集的详细数据

### 步骤2: 填写数据

#### 基本设置工作表

| 单元格 | 说明 | 示例 |
|--------|------|------|
| B2 | X轴数据(类别) | 0,2.5,5,7.5 |
| B3 | 图表类型 | 折线图 |
| B4 | 图表标题 | 多数据集示例 |
| B5 | X轴标签 | 葡萄糖浓度 (g/L) |
| B6 | Y轴标签 | DCW (g/L) |
| B7 | 显示网格 | 是/否 |
| B8 | 显示图例 | 是/否 |
| B9 | 显示数值 | 是/否 |

**图表类型可选值:**
- 折线图
- 柱状图
- 散点图
- 误差棒图
- 直方图
- 饼图
- 雷达图

#### 数据集工作表

| 列 | 说明 | 必填 |
|----|------|------|
| A | 数据集标签 | 是 |
| B | Y轴数据(逗号分隔) | 是 |
| C | 误差数据(逗号分隔) | 否 |
| D | 颜色 | 否 |

**颜色可选值:**
- blue (蓝色)
- orange (橙色)
- green (绿色)
- red (红色)
- purple (紫色)
- brown (棕色)
- pink (粉色)
- cyan (青色)
- yellow (黄色)
- black (黑色)

**数据格式示例:**
```
A列(标签)    B列(Y轴数据)              C列(误差数据)              D列(颜色)
0             0.2,1.1,0.95,0.55         0.05,0.1,0.08,0.06        blue
2.5           0.5,1.15,0.98,0.6         0.06,0.12,0.09,0.07       orange
5             0.48,1.0,0.96,0.58        0.05,0.11,0.08,0.06       green
7.5           0.45,0.95,0.94,0.56       0.04,0.1,0.07,0.05        red
```

### 步骤3: 导出JSON配置

1. 填写完所有数据后,按 `Alt + F8` 打开宏对话框
2. 选择 `ExportToJSON` 函数
3. 点击 `运行`
4. 在弹出的文件对话框中选择保存位置和文件名
5. 点击 `保存`

```json

```

## 导入到Python绘图工具

1. 打开Python绘图工具
2. 点击 `📥 导入配置` 按钮
3. 选择刚才导出的JSON文件
4. 所有设置将自动加载
5. 点击 `🚀 一键绘制图表` 即可生成图表

## 注意事项

1. **数据格式**: 所有数值数据必须用逗号分隔,不要使用空格或其他符号
2. **图表类型**: 确保图表类型名称准确,支持中文
3. **误差数据**: 某些图表类型(如误差棒图)必须填写误差数据
4. **数据长度**: Y轴数据的长度必须与X轴数据的长度一致(饼图和直方图除外)
5. **颜色设置**: 如果不指定颜色,系统将使用默认颜色

## 快捷键设置(可选)

在宏代码中取消注释以下行可以设置快捷键:

```javascript
Application.OnKey("^e", "ExportToJSON");   // Ctrl+E 导出JSON
Application.OnKey("^t", "CreateTemplate"); // Ctrl+T 创建模板
```

## 常见问题

### Q: 运行宏时提示"缺少必要的工作表"?
A: 请先运行 `CreateTemplate` 创建必要的工作表。

### Q: 导出的JSON文件乱码?
A: 确保使用UTF-8编码保存,宏代码已经设置为Unicode编码。

### Q: 数据导入后图表显示不正确?
A: 检查数据格式是否正确,特别是逗号分隔符和数据长度。

### Q: 如何添加更多数据集?
A: 在"数据集"工作表中继续添加行,每个数据集占一行。

## 技术支持

如有问题,请检查:

2. 宏安全设置是否允许运行宏
3. 数据格式是否符合要求
