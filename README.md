# PyPlot 绘图工具

一个用户友好的 GUI 工具，用于使用 Matplotlib 创建各种类型的图表。无需编码！

## 功能特点

- **多种图表类型**：折线图、柱状图、散点图、误差棒图、直方图、饼图和雷达图
- **多数据集支持**：支持 1-7 个数据集，可自定义标签
- **误差棒**：为数据添加误差棒
- **自定义设置**：标题、标签、网格、图例和数值显示选项
- **用户友好界面**：直观的 GUI 界面，带示例数据
- **数值显示**：可选择在图表上显示精确数值
- **颜色控制**：为每个数据集单独设置颜色
- **灵活的输入格式**：支持逗号、空格或换行符分隔的数据输入

## 如何使用

1. **选择数据集数量**：选择要绘制的数据集数量（1-7）
2. **输入 X 轴数据**：输入用逗号、空格或换行符分隔的类别名称
3. **输入数据集数据**：为每个数据集输入：
   - 标签：数据集的名称
   - Y 轴数据：用逗号、空格或换行符分隔的数值
   - 误差数据：可选的误差值，用逗号、空格或换行符分隔
   - 颜色：为数据集选择颜色
4. **图表设置**：
   - 图表类型：选择您想要的图表类型
   - 标题：输入图表的标题
   - X 轴标签：X 轴的标签
   - Y 轴标签：Y 轴的标签
   - 显示网格：勾选以显示网格线
   - 显示图例：勾选以显示图例
   - 显示数值：勾选以在图表上显示精确数值
5. **绘制图表**：点击 "🚀 一键绘制图表" 按钮生成图表
